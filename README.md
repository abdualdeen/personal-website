# personal-website
repo for my personal website.

### Quick Start

run:

```
npm i
```

and to start the server,

```
npm run dev
```

or 

```
vite
```

### Tech Stack 

* ReactJS 
* Vite
* TypeScript
* Tailwind for CSS
* ChartsJS
* Firebase

### Build and Deploy

To build the project:

```
npm run build
```

then to emulate running the built project for testing before deployment:

```
firebase emulators:start
```

to deploy:

```
firebase deploy
```
